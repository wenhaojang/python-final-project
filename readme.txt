环境： python 3.13.5
所用的包：numpy,pandas,matplotlib,scipy,statsmodels,openpyxl,statsmodels,scikit-learn

data.ipynb文件为处理原始数据至最后分析用数据的脚本。
EDA.ipynb为文中EDA（数据探索式分析）部分的脚本
分析.ipynb即为文中所有建模分析的代码。
应该都是能直接运行的（当时是全用的本地路径，感觉是挨个改完了的，如果报错应该是有漏的路径没改好……）

原始数据文件夹中为所有收集的非文本的原始数据（ETF资金流量、百度指数、控制变量）。
文本数据与处理代码文件夹为收集的文本原始数据，以及处理它们为权重或打分的代码。
新数据为处理后的数据：分行业的政策权重、评论文本的打分、各个已经处理为变量的值。
整合数据首先包含整合新数据中处理好的数据用于分析的表格，还有所有的分析的结果。

其余文件为tex形式的报告编译所配套的文件。
